
[摆动排序](https://leetcode.com/problems/wiggle-sort-ii/)

给定数组`nums`，重排数组使得`nums[0] < nums[1] > nums[2] < nums[3]...`

```
Example 1:

Input: nums = [1, 5, 1, 1, 6, 4]
Output: One possible answer is [1, 4, 1, 5, 1, 6].

Example 2:

Input: nums = [1, 3, 2, 2, 3, 1]
Output: One possible answer is [2, 3, 1, 3, 1, 2].
```

可以假设所有case都有解

Follow Up：能否做到时间复杂度$O(n)$或者空间复杂度$O(1)$
